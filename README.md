# contribution-leaderboard

## To start the server:
- Create .env file with key DBURL (mongoose DB URL)
<br>
```
npm i
npm run serve
```
<br>
<br>

## To add it as a webhook:
- Create new webhook
- Paste https://api.contributor.kuosc.np/api/v1/webhook/post as link and select send me all events
- Save Changes
